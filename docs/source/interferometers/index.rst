Interferometers
===============

The interferometers module is designed for enabling the simulation of circuits as part of imperfect interferometric systems. It features a mapper which decomposes and configures a provided circuit into an interferometer, and :doc:`../interferometers_reference/error_model` for the inclusion of various imperfections into the system.
